# dart-lsp

> **Note:** Temporary plugin until an official `dart-lsp` is added to [claude-plugins-official](https://github.com/anthropics/claude-plugins-official).

Dart/Flutter language server for Claude Code, providing code intelligence features like go-to-definition, find references, hover information, and error checking.

## Supported Extensions
`.dart`

## Installation

The Dart language server is bundled with the Dart SDK — no separate install needed.

Verify it's available:

```bash
dart language-server --help
```
